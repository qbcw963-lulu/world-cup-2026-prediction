# 2026 世界杯预测模型方法

## 目标市场

- 胜平负：主胜、平局、客胜概率。
- 比分：概率最高的正确比分，以及完整比分分布。
- 让球胜平负：按指定让球线重新聚合比分概率。
- 半全场：通过上半场和全场联合进球分布计算九种结果。

## 第一版模型

第一版采用可解释的独立 Poisson 进球模型。球队初始强度经过以下因素修正后，得到双方预期进球：

1. 赛前球队强度：Elo、FIFA 排名、近两年正式比赛和近期状态。
2. 阵容：首发质量、伤停、停赛、轮换、关键位置替代水平。
3. 场地：东道主优势、旅途、海拔、草皮和球场类型。
4. 赛程：休息天数、加时赛历史负荷和跨时区旅行。
5. 天气：温度、湿度、风速和降水对比赛节奏的影响。
6. 裁判：历史黄牌、红牌、点球、犯规和主客判罚倾向。
7. 战意：出线形势、净胜球需求、是否接受平局以及轮换概率。
8. 市场：去除庄家水位后的隐含概率，仅用于校准和爆冷定义。单一博彩公司
   完整三项开盘赔率当前使用 25% 权重；没有完整三项时权重为 0。

## 爆冷定义

爆冷必须按赛前概率定义，不能按赛后观感倒推：

- 重大爆冷：实际结果的赛前概率低于 20%。
- 中等爆冷：20%–30%。
- 轻度爆冷：30%–40%。
- 正常结果：实际结果概率不低于 40%。

在缺少可靠赛前赔率时，系统使用冻结的赛前强度模型概率，并将
`classification_basis` 标成 `model_only`。取得赔率后改为
`market_consensus`，避免混淆两种口径。

## 数据纪律

- 每条外部信息必须保留来源 URL、抓取时间和验证状态。
- 不确定的助理教练、球员、裁判和伤停不得猜测，使用
  `pending_verification`。
- 比赛开赛后冻结赛前特征，禁止用赛中或赛后信息污染预测。
- 预测是概率估计，不保证命中，也不构成投注建议。
